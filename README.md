# DreamCockpit (CPT)

## Features
This DreamOs plugin installs ALL DreamCockpit series plugins:
- MediaCockpit (pictures, movies, slideshows, and more...)
- MovieCockpit (recordings database and more...)
- PiconCockpit (download channel picons)
- PowerCockpit (complete recordings, archiving of recordings, file operations in idle mode after pressing power-off))
- TimeshiftCockpit (on demand and permanent timeshift)
- TMDBCockpit (TMDB movie infos)

## Installation
To install DreamCockpit execute the following command in a console on your dreambox:
- apt-get install wget (required the first time only)
- wget https://dream-alpha.github.io/DreamCockpit/dreamcockpit.sh -O - | /bin/sh

The installation script will also install a feed source that enables a convenient upgrade to the latest version with the following commands or automatically as part of a DreamOS upgrade:
- apt-get update
- apt-get upgrade
